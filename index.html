6.2 Properties of Sinusoidal Functions

Key Vocabulary PERIODIC: a function that repeats at regular intervals (period T)
SINUSOIDAL: a periodic function that looks like a smooth wave (sine or cosine shape)
AMPLITUDE (a): half the distance between max and min ‚Üí a = (max ‚ąí min) / 2
PERIOD (T): horizontal length of one complete cycle
AXIS OF SYMMETRY (c): midline of the wave ‚Üí c = (max + min) / 2
RANGE: from (c ‚ąí a) to (c + a)

Identifying Periodic vs Sinusoidal from Graphs

Rules

Periodic: pattern repeats ‚Üí YES. If pattern never repeats ‚Üí NO (e.g. always increasing).

Sinusoidal: periodic AND smooth wave (no sharp corners, no flat tops) ‚Üí YES.

Graph a): smooth repeating wave ‚Üí periodic AND sinusoidal ‚úď

Graph b): repeating but zigzag/triangle ‚Üí periodic but NOT sinusoidal

Graph c): always increasing trend ‚Üí NOT periodic

Graph d): smooth repeating wave ‚Üí periodic AND sinusoidal ‚úď

Reading Real-World Graphs Period ‚Üí how long one full cycle takes
Amplitude ‚Üí how far the value swings above/below the midline
Axis ‚Üí the average/equilibrium value
Frequency ‚Üí how many cycles per unit time (= 1/Period)

Key Differences: sin vs cos

Comparison

cos(x) = sin(x + 90¬į) ‚Üí cosine is sine shifted LEFT 90¬į

sin(x) = cos(x ‚ąí 90¬į) ‚Üí sine is cosine shifted RIGHT 90¬į

sin is ODD function: sin(‚ąíx) = ‚ąísin(x)

cos is EVEN function: cos(‚ąíx) = cos(x)

6.5 Transformations of Sinusoidal Functions

General Form: f(x) = a¬∑sin(k(x ‚ąí d)) + c OR f(x) = a¬∑cos(k(x ‚ąí d)) + c |a|: AMPLITUDE (vertical stretch); a<0 ‚Üí reflected in x-axis
k: determines PERIOD ‚Üí Period = 360¬į/|k|
d: PHASE SHIFT (horizontal); d>0 ‚Üí shift RIGHT; d<0 ‚Üí shift LEFT
c: VERTICAL SHIFT ‚Üí axis of symmetry = y = c
Range: {c‚ąí|a| ‚Č§ y ‚Č§ c+|a|}
Domain: always {x ‚ąą ‚ĄĚ}

Example 1 ‚ÄĒ f(x) = 3sin(2(x ‚ąí 60¬į)) + 4

Identify all parameters

a = 3 ‚Üí amplitude = 3 (vertical stretch √ó3)

k = 2 ‚Üí period = 360¬į/2 = 180¬į

d = 60¬į ‚Üí phase shift 60¬į RIGHT

c = 4 ‚Üí axis y = 4; shift UP 4

Max = 4+3 = 7; Min = 4‚ąí3 = 1; Range: {1 ‚Č§ y ‚Č§ 7}

Key points of base sin shifted: start 60¬į, max at 60¬į+45¬į=105¬į, back to axis 60¬į+90¬į=150¬į, min 60¬į+135¬į=195¬į, end cycle 60¬į+180¬į=240¬į

Example 2 ‚ÄĒ y = ‚ąí2cos(3x) ‚ąí 1

Graph and state properties

a = ‚ąí2: amplitude = 2, reflected in x-axis (starts at MIN not max)

k = 3: period = 360¬į/3 = 120¬į

d = 0: no phase shift

c = ‚ąí1: axis y = ‚ąí1; shift DOWN 1

Max = ‚ąí1+2 = 1; Min = ‚ąí1‚ąí2 = ‚ąí3; Range: {‚ąí3 ‚Č§ y ‚Č§ 1}

Phase shift = 0 (no horizontal shift)

Key start point (0¬į): y = ‚ąí2cos(0)‚ąí1 = ‚ąí2(1)‚ąí1 = ‚ąí3 (starts at minimum because reflected)

How to Sketch ‚ÄĒ Step Method

Systematic Approach

1. Identify a, k, d, c from equation (factor k out of argument first!)

2. Draw axis y = c as dashed line

3. Mark amplitude above and below axis

4. Period = 360¬į/|k|; divide into 4 equal parts (quarter-period = T/4)

5. Apply phase shift d to key points

6. For sin: 0‚Üímax‚Üí0‚Üímin‚Üí0; For cos: max‚Üí0‚Üímin‚Üí0‚Üímax (or reflected if a<0)

7. Connect smoothly; label axis, amplitude, period on graph

6.6 & 6.7 Determining Equations & Applications

Algorithm: Graph ‚Üí Equation Step 1: Find max and min from graph
Step 2: a = (max ‚ąí min) / 2
Step 3: c = (max + min) / 2 [axis of symmetry]
Step 4: Period T from graph ‚Üí k = 360¬į/T
Step 5: Phase shift d ‚Üí where does the first max (cos) or ascending zero (sin) occur?
Step 6: Write f(x) = a¬∑sin(k(x‚ąíd))+c OR a¬∑cos(k(x‚ąíd))+c
Step 7: Verify with known points

Vocabulary

  • Period: length of one full cycle
  • Amplitude: (max‚ąímin)/2
  • Axis: y = (max+min)/2 = c
  • Phase shift: horizontal translation = d
  • Range: [c‚ąía, c+a]
  • Sinusoidal: periodic smooth wave

General Form

f(x) = a¬∑sin(k(x ‚ąí d)) + c

  • a: amplitude; a<0 ‚Üí x-axis reflection
  • k: Period = 360¬į/|k|
  • d: phase shift; d>0 ‚Üí RIGHT
  • c: axis y=c; vertical shift

Base Graph Key Points

  • sin starts at 0 going UP

Graph ‚Üí Equation Algorithm

  • 1. Find max & min
  • 2. a = (max‚ąímin)/2
  • 3. c = (max+min)/2
  • 4. T from graph ‚Üí k = 360¬į/T
  • 5. d = where cycle starts
  • 6. Use cos if max at start
  • 7. Use sin if zero (ascending) at start
