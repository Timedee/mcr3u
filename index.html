#1b Is 5, 15, 45, 135, … geometric? If so, state the general term. ▼
Given
Sequence: 5, 15, 45, 135, …
Check consecutive ratios: 15/5 = 3, 45/15 = 3, 135/45 = 3
Identify values
Common ratio is constant ∴ sequence is geometric
a = 5, r = 3
General Term
tₙ = ar^(n−1) = 5 × 3^(n−1)
#1d Is 6000, 3000, 1500, … geometric? General term? ▼
Given
Sequence: 6000, 3000, 1500, …
Check: 3000/6000 = 1/2, 1500/3000 = 1/2
Identify values
Ratio is constant ∴ geometric
a = 6000, r = 1/2
General Term
tₙ = 6000 × (1/2)^(n−1)
#2b For 625, 1250, 2500, … find general term. ▼
Given
a = 625, r = 1250/625 = 2
General Term
tₙ = 625 × 2^(n−1)

📘 Application Problems ▼
#11 To get from t₅ to t₈: t₈/t₅ = r³. Given t₅ = 45, t₈ = 360. Find r and t₂₀. ▼
Given
t₅ = 45, t₈ = 360
Find r
To get from t₅ to t₈, multiply by r three times:
t₈/t₅ = r³
360/45 = 8 = r³
r = ∛8 = 2
Find a
t₅ = ar⁴ → 45 = a(2)⁴ = 16a → a = 45/16
t₂₀
t₂₀ = (45/16) × 2^19 = 45 × 2^15 = 45 × 32768 = 1 474 560
#14 Bacteria: f(1) = 5120. Each observation multiplied by 3/2. Find f(9). ▼
Given
a = 5120, r = 3/2
General Term
f(n) = 5120 × (3/2)^(n−1)
f(9) — 9th observation
f(9) = 5120 × (3/2)^8 = 5120 × 6561/256 = 131 220 bacteria
#15 Given t₅ = 45 and t₇ = 360. Find r using ratio method, then find t₂₉. ▼
Given
t₅ = 45, t₇ = 360
Find r
t₇/t₅ = r²
360/45 = 8 = r²
r = ±√8 = ±2√2
Taking the square root: r = √8 (take positive for this context)
Find r from solutions manual context
Actually from t₅ to t₇ is 2 steps: t₇ = t₅ × r²
r² = 360/45 = 8 → r = ±2√2
t₂₉ using the known values
From t₇ to t₂₉ is 22 steps:
t₂₉ = t₇ × r²² = 360 × (8)^11 = 360 × 8 589 934 592
Use the general term once a and r are pinned down from the specific sequence.
#17 Similarities and differences between arithmetic and geometric sequences. ▼
Similarities
• Both can be defined by a general term that depends on only two quantities (the first term and the common ratio or difference).
• Both can be defined recursively.
• Both are discrete functions.
Differences
• Arithmetic: add a constant difference (d); general term is linear: tₙ = a + (n−1)d
• Geometric: multiply by a constant ratio (r); general term is exponential: tₙ = ar^(n−1)
Key Test
Subtract consecutive terms → constant? → Arithmetic
Divide consecutive terms → constant? → Geometric

t₁₀

General Term

t₁₀

#6 Art piece: initial $10 000, decreases 10% per week. After how many weeks ≤ $100? ▼
Given
Price decreases 10% → multiplied by 0.9 each week.
After 1 week: $10 000 × 0.9 = $9 000. So a = 9000, r = 0.9
General Term
tₙ = 9000 × 0.9^(n−1)
Solve tₙ ≤ 100
By substituting integer values of n:
Check n = 43: 9000 × 0.9^42 ≈ 107.75
Check n = 44: 9000 × 0.9^43 ≈ 96.98
Answer
n = 44 is the smallest value → after 44 weeks, price ≤ $100.
